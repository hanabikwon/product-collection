DIFF

시간감각 조정 타이머 — Expected vs Actual

Problem

"30분이면 될 거 같은데..." → 2시간 걸림. 시간 예측이 계속 틀리면 일정이 꼬이고 자책하게 된다.

Solution

예상 시간과 실제 시간을 기록하고 비교하여 자신만의 시간감각 패턴을 분석. Faster/Slower/Accurate로 과소평가/과대평가 경향을 파악.

Concept

시간의 명확성을 최우선으로. 블랙 모노톤 베이스에 빨강/파랑/초록으로 차이를 시각화하여 직관적으로 인지하게 설계.

Solo · 기획 → 디자인 → 개발 → 배포

KEY FEATURES

  • 예상/실제 시간 비교 타이머 (최대 12시간)
  • Faster/Slower/Accurate 자동 판정
  • 카테고리별 통계 및 편향 분석
  • Google 로그인 & 클라우드 백업

2025 연말결산 — 한 해를 천천히 돌아볼 시간

Problem

한 해가 끝나갈 때, 막연히 지나간 시간을 정리하고 싶지만 어디서부터 시작해야 할지 모르겠다.

Solution

20개의 질문으로 한 해를 돌아보고, AI가 나만의 이야기로 정리해주는 연말결산 테스트.

Concept

바느질한 집, 편지와 우체통, 눈 내리는 효과. 하트로 만족도 표시. 따뜻한 핸드메이드 무드로 한 해를 포근하게 마무리.

Solo · 기획 → 디자인 → 개발 → 배포

KEY FEATURES

  • 20문항 연말 회고 질문
  • AI 분석 (총평, 감정, 성장, 취향, 응원)
  • 결과 공유 (카카오톡, 링크)
  • 이미지 저장

Recipesoup

요리하는 사람을 위한 감성 레시피 아카이빙 앱

Problem

정성스레 만든 요리를 나만의 레시피로 기록하고 싶은데, 여기저기 흩어져서 찾기 어렵고 정리하기도 번거롭다.

Solution

AI로 쉽게 기록하고, 흩어진 레시피를 한 곳에 모으고, 요리하는 마음도 함께 담는 감성적인 UX.

Concept

토끼 세계관. 견습생에서 시작해 경험을 쌓고, 시그니처 요리를 만들고, 셰프가 되어 나만의 레스토랑을 여는 32단계 성장 스토리.

Solo · 기획 → 디자인 → 개발 → 배포

KEY FEATURES

  • 퀵레시피: 요리명만 입력하면 끝
  • 냉장고 털기: 남은 재료로 추천
  • 링크 가져오기: 블로그 레시피 요약
  • 사진 분석: 완성 요리 역추적
  • 토끼굴: 32단계 게이미피케이션 성장 시스템

Flutter Supabase OpenAI API Vercel
