시간감각 조정 타이머 — Expected vs Actual

Problem

"30분이면 될 것 같은데" 시작한 작업이 2시간이 걸린다. 시간 예측 오류는 반복되지만, 측정 데이터 없이는 자신의 편향 패턴을 인식할 수 없고 개선되지 않는다.

Solution

작업의 예상-실제 시간 차이를 측정하여 Faster/Slower/Accurate로 과소평가/과대평가 경향을 파악. 개인의 추정 편향을 정량화하고, 패턴 인식과 예측 개선을 가능하게 한다.

Concept

시간의 명확성을 최우선으로. 블랙 모노톤 베이스에 빨강/파랑/초록으로 차이를 시각화하여 직관적으로 인지하게 설계.

Solo · 기획 → 디자인 → 개발 → 배포

Key Features

  • 예상/실제 시간 비교 타이머 (최대 4시간)
  • Faster/Slower/Accurate 자동 판정
  • 카테고리별 통계 및 편향 분석
  • Google 로그인 & 클라우드 백업

Recipesoup

요리하는 사람을 위한 감성 레시피 아카이빙 앱

Problem

정성스레 만든 요리를 나만의 레시피로 기록하고 싶은데, 여기저기 흩어져서 찾기 어렵고 정리하기도 번거롭다.

Solution

AI로 쉽게 기록하고, 흩어진 레시피를 한 곳에 모으고, 요리하는 마음도 함께 담는 감성적인 UX.

Concept

토끼 세계관. 견습생에서 시작해 경험을 쌓고, 시그니처 요리를 만들고, 셰프가 되어 나만의 레스토랑을 여는 32단계 성장 스토리.

Solo · 기획 → 디자인 → 개발 → 배포

Key Features

  • 퀵레시피: 요리명만 입력하면 끝
  • 냉장고 털기: 남은 재료로 추천
  • 링크 가져오기: 블로그 레시피 요약
  • 사진 분석: 완성 요리 역추적
  • 토끼굴: 32단계 게이미피케이션 성장 시스템

Tech Stack

Flutter Supabase OpenAI API Vercel
